
# Proyecto Final Python - Nivel Inicial

Este repositorio contiene dos aplicaciones sencillas desarrolladas en Python como parte de un proceso de aprendizaje de programación.

---

## 1. Sistema de Gestión de Tickets
Una aplicación de consola (CLI) diseñada para organizar y gestionar consultas de soporte técnico.

### Características:
- **Gestión de Alta:** Permite ingresar nombre, sector, asunto y problema.
- **Identificación:** Asigna automáticamente un número de ticket único (aleatorio entre 1000-9999).
- **Consulta:** Búsqueda rápida de tickets existentes mediante su número identificador.
- **Interactividad:** Menú intuitivo que permite realizar múltiples operaciones sin cerrar el programa.

---

## 2. Conversor de Unidades de Longitud
Una herramienta con interfaz gráfica (GUI) creada con **Tkinter**, ideal para realizar conversiones rápidas entre unidades de medida.

### Características:
- **Interfaz Amigable:** Selección de unidades mediante menús desplegables.
- **Flexibilidad:** Permite convertir entre Metros, Centímetros, Pulgadas y Pies.
- **Validación:** Incluye control de errores para asegurar que solo se procesen números positivos y datos válidos.
- **Precisión:** Cálculos automatizados basados en un sistema de conversión universal a metros.
